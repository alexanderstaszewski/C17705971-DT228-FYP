Touhou Character Identifier

This project aims to identify a handful of characters from Touhou 06 using a machine learning and deep learning approach.

To see which characters are present in an image upload a file below (or drag and drop into the box) and click the "Detect Faces" button. If you wish to use this without classifying characters then it is possible to enable the "Ignore character identification" checkbox to disable this feature.

The characters supported by this project are:

  • Reimu Hakurei
  • Marisa Kirisame
  • Remilia Scarlet
  • Flandre Scarlet
  • Sakuya Izayoi
